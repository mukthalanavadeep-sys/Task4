📊 Superstore Sales Analytics Dashboard
📌 Project Overview

This project is an interactive data visualization dashboard built using Tableau Public on the Sample Superstore dataset.

It provides insights into:

Sales performance
Profit analysis
Product trends
Regional performance
Discount impact on profit

The goal is to understand business performance and support data-driven decision making.

📌 Dashboard Insights
📅 Sales Trend
Sales are stable across months
Peak sales observed from September to November
Seasonal buying behavior detected
🌍 Sales by Region
West region: Highest sales
East region: Second highest
Central region: Moderate performance
South region: Lowest sales
🏆 Top Products
HP and office electronics are top contributors
A small number of products generate major revenue
💵 Profit Analysis
Technology category generates highest profit
Furniture shows lowest profitability
Profit is uneven across categories
🎯 Discount vs Profit
Higher discounts reduce profit
Optimal discount range: 0.05 – 0.15
Shows strong relationship between pricing strategy and profit
