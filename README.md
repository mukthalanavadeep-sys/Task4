# Task 4 - Dashboard Design

## Objective
Built an interactive dashboard using Power BI/Tableau for sales analysis.

## Tools Used
- Power BI / Tableau
- Excel / Kaggle Dataset

## Features
- KPI cards (Sales, Profit, Quantity)
- Time-series analysis
- Interactive filters (Slicers)
- Region and category analysis

## Insights
- Technology category performs best
- Profit varies by region
- Discounts impact profit negatively

## Output
Interactive dashboard + PPT summary
